# Apps_Script (Selenium / xPath)
디지털 렌딩 파트너 사이트에서 당사가 적합하게 표시되는지 확인
  - Selenium 기반 파트너 사이트 변경 감지 자동화 스크립트 개발
  - DOM 구조 변화 및 XPath 변경 자동 감지 → UI 안정성 및 규격(Compliance) 유지
  - Jenkins CI 파이프라인에 통합하여, 빌드 발생 시 자동 실행되도록 구성
